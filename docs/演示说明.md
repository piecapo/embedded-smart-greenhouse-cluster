# 演示说明

## 1. 编译 C++ 程序

```powershell
cmake -S . -B build
cmake --build build
.\build\Debug\greenhouse_cluster.exe 5
```

## 2. 可选启动网页服务

在项目目录运行：

```powershell
python server/app.py
```

浏览器打开：

```text
http://127.0.0.1:8080
```

## 3. 演示流程

1. 先运行 C++ 程序，展示 6 个蔬菜大棚状态。
2. 观察每轮温度、湿度、土壤湿度、光照、CO2 的变化。
3. 当出现高温、缺水、光照不足等情况时，程序输出预警。
4. 展示 `include/` 和 `src/`，说明 C++ 类设计。
5. 展示 `firmware/esp32_greenhouse_node`，说明真实硬件接入方式。
6. 如需可视化，再打开网页看板演示。

## 4. 答辩介绍词

本项目面向设施农业中的蔬菜大棚集群管理场景，使用 C++ 实现大棚节点建模、环境数据模拟、自动控制和异常预警。真实硬件端可使用 ESP32 作为嵌入式控制节点，采集环境数据并控制执行设备。即使没有真实硬件，也可以通过 C++ 仿真程序展示完整的数据闭环，方便教学演示和后续扩展。
